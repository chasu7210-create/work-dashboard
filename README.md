# 📊 개인 업무 대시보드

바이브코딩으로 만든 개인용 업무관리 도구입니다.  
업무분장 · 사업목록 · 예산현황 · 업무매뉴얼 · 메모를 한 화면에서 관리합니다.

---

## ⬇️ 다운로드

아래 링크를 클릭하면 HTML 파일이 열립니다.  
브라우저에서 **우클릭 → 다른 이름으로 저장** 하거나,  
**Ctrl+S** 로 저장하세요.

🔗 [work-dashboard.html 다운로드](https://raw.githubusercontent.com/chasu7210-create/work-dashboard/main/work-dashboard.html)

---

## 📋 주요 기능

- **홈** — 업무·사업·예산 요약 카드 + 빠른 메모
- **업무분장** — 담당 업무 목록 관리 (주담당/보조/지원)
- **사업목록** — 진행 사업 상태 관리 (계획중/진행중/완료)
- **예산현황** — 사업별 예산 집행률 확인
- **업무매뉴얼** — 자주 쓰는 절차 메모
- **메모(노트)** — 카드형 메모, 색상·핀 고정 지원

---

## 💡 사용 방법

1. 위 링크에서 파일 다운로드 (`Ctrl+S`)
2. 다운로드한 `work-dashboard.html`을 **Chrome 또는 Edge**로 열기
3. 데이터는 브라우저 localStorage에 자동 저장됨
4. 정기적으로 **사이드바 하단 → JSON 백업** 권장
